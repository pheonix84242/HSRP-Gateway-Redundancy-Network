# HSRP Gateway Redundancy Network

## Overview
This project demonstrates First Hop Redundancy using HSRP (Hot Standby Router Protocol). Two routers are configured to provide a highly available default gateway for end devices. If the active router fails, the standby router automatically takes over, ensuring uninterrupted network connectivity.

## Technologies Used
- Cisco Packet Tracer
- HSRP
- Routing & Switching
- Gateway Redundancy
- Network High Availability

## Network Topology
- 2 Routers (R1, R2)
- 1 Switch
- 4 PCs
- 1 Server
- HSRP Virtual Gateway: 192.168.10.1

## Features
- Active and Standby HSRP Routers
- Automatic Gateway Failover
- Preemption Configuration
- High Availability Design
- Enterprise Network Redundancy

## Skills Demonstrated
- First Hop Redundancy Protocol (FHRP)
